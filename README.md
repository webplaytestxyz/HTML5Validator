# HTML5Validator

A **desktop application** for auditing websites for HTML5 compliance and SEO essentials using Selenium and BeautifulSoup.

---

## Features

* Automated website scanning with Selenium
* HTML5 validation (DOCTYPE, `<html lang>` attribute)
* SEO checks (title, H1, canonical tags, robots meta)
* Image alt text validation
* OpenGraph and Twitter Cards detection
* Website screenshot preview
* Report export to TXT

---

## Requirements

* Python 3.x
* Selenium
* BeautifulSoup4
* Pillow
* Chrome or Chromium browser

---

## Usage

Run the application locally:

```bash
python html5.py
```

**Important Notes / Terms of Use:**

* This application **runs locally on your computer** and uses **your own internet connection** to fetch and analyze websites.
* The app **does not store or send data anywhere**.
* Only scan websites you have permission to access. Avoid heavy or repeated scans of large websites to prevent overloading their servers.
* For safe demo purposes, you can use websites explicitly designed for learning/scraping, such as:

  * [books.toscrape.com](https://books.toscrape.com/)
  * [quotes.toscrape.com](https://quotes.toscrape.com/)
  * [scrapethissite.com](https://scrapethissite.com/)
